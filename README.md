# Web Rust Car

> RC 玩具车，使用 Web 浏览器作为遥控器，ESP32C3 作为主控芯片，Rust 语言实现主控逻辑。

## 如何使用

任意找一台支持蓝牙的手机，打开浏览器访问 [todo]，根据界面提示找到 Web-Rust-Car 的蓝牙，链接后开始畅玩。
